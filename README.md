###Timo是什么

- **Timo**是一个分布式[MySQL] (https://www.mysql.com)数据库中间件。^(#｀∀´)_Ψ
- **Timo**源于[Cobar] (https://github.com/alibaba/cobar)，并在[Cobar] (https://github.com/alibaba/cobar)的基础上进行改造。Ｏ(≧▽≦)Ｏ
- **Timo**目前是实验性质，主要用于熟悉相关技术，实现个人想法，提高姿势水平。(●｀･(ｴ)･´●)
